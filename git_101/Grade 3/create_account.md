Before you can share your projects online or collaborate with others, you need a GitHub account. GitHub is the most popular platform for hosting Git repositories and working with teams. Other platsforms like GitLab and Bitbucket also exist, but GitHub is the most widely used.


## Steps to Create a GitHub Account

1. **Visit GitHub**: Go to [github.com](https://github.com).
2. **Sign Up**: Click on the "Sign up" button.
3. **Enter Your Details**: Provide a username, email address, and password.
4. **Verify Your Account**: Follow the instructions to verify your email address.
5. **Choose a Plan**: You can start with the free plan, which is sufficient for most users.
6. **Complete Your Profile**: Add a profile picture and bio to personalize your account.

## Set Up Basic Profile Settings

1. **Profile Picture**: Upload a clear profile picture to help others recognise you.
2. **Bio**: Write a short bio to introduce yourself to the GitHub community.
3. **Location**: Optionally, add your location to connect with local developers.

## Multi Factor Authentication (MFA)

GitHub supports Multi-Factor Authentication (MFA) to enhance security. It is highly recommended to enable MFA for your account.

1. **Go to Settings**: Click on your profile picture in the top right corner and select "Settings".
2. **Security**: Navigate to the "Security" section.
3. **Enable MFA**: Follow the instructions to set up MFA using an authentication app or SMS.
